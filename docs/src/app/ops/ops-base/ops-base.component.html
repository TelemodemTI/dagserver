SSH Operator

This operator allows you to execute a command through a connection via SSH to the server. Supports private key and know host

Icon

Param Dialog




Host: hostname of target machine

User: username

Port: portnumber

knowhost file: file that is used to store the public key of the server to which the operator will connect.

Useful Tip:

The ssh-keyscan -t rsa #ip command is used to scan and display the RSA public key of the SSH server at the specified IP address (#ip). This public key can be manually added to the "knownhosts" file on your local system, allowing secure SSH connections without key confirmation prompts in the future. It is useful for automating the setup of SSH connections to known servers.

private key file: ppk file for SSH authentication.

Useful Tip:

PPK (PuTTY Private Key) files are used in the PuTTY SSH client environment and are used to authenticate to SSH servers using private keys instead of passwords.

File Operator

This operator allows you to read or write the content of a structured flat file (CSV) to or from the xcom of the DAG

Icon

Param Dialog




Mode: read or write

Filepath: location of file

Row Delimiter: delimiter of row

Useful Tip:

If an empty row delimiter is provided, the file content will be treated as unstructured. This means that the content associated with the read/write will be passed directly from/to the corresponding XCOM.

First Row Titles: first row is column titles.

Xcom: XCOM that will be used in the case of writing mode, to obtain the data.

JDBC Operator

This operator allows you to execute a SQL command in a JDBC compatible RDMS

Icon

Param Dialog




Url: database host

User: database user

Pwd: database pwd

Driver: database driver.

Driver Path: location of driver files

XCOM: XCOM that will be associated with the SQL query to be executed.

Useful Tip:

If it is required to interact with data from an XCOM resulting from the execution of another node, within the DAG, the use of named parameters in the queries is recommended

Mail SMTP Operator

This operator is used to send an email using an SMTP server.

Icon

Param Dialog




Host: host SMTP server

Port: port SMTP server

User Smtp: user SMTP server

Pwd Smtp: pwd SMTP server

From Mail: from Mail

To Mail: Target mail

Subject: Subject

XCOM: XCOM with the body if aplicable

Useful Tip:

In the case of this stage, in the script tab, you can write the body text of the email directly, if you do not want to use the message from another resulting XCOM
